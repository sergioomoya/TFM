# Resumen

La detección de fraude con tarjetas de crédito (CCFD) es un problema difícil, que requiere analizar grandes volúmenes de datos de transacciones para identificar patrones de fraude. Los grandes volúmenes de datos, junto con las técnicas en evolución de los estafadores, hacen imposible que los investigadores humanos aborden este problema de manera eficiente. En la última década, CCFD se ha complementado cada vez más con algoritmos informáticos conocidos como *Aprendizaje Automático* (ML), que permiten buscar y detectar patrones a partir de grandes cantidades de datos. Se ha demostrado que los algoritmos de ML mejoran significativamente la eficiencia de los sistemas de detección de fraude y ayudan a los investigadores de fraude a detectar transacciones fraudulentas.

ML para CCFD se ha convertido en un campo de investigación activo. Esto se ilustra con un gran número de publicaciones sobre el tema y sus prometedoras afirmaciones. Al mismo tiempo, se debe tener precaución al interpretar los resultados de estas publicaciones. Como se señaló en {ref}`ML_For_CCFD`, el campo de ML para CCFD es relativamente nuevo y no existen puntos de referencia reconocidos, ni metodologías, para comparar y evaluar las técnicas propuestas {cite}`priscilla2019credit,mekterovic2018systematic,zojaji2016survey`.

Este libro tiene como objetivo dar un primer paso en esta dirección, proponiendo implementaciones reproducibles y metodologías de referencia para abordar el tema de ML para CCFD. Las implementaciones y metodologías propuestas se basan en nuestra experiencia en el campo, siguiendo nuestra colaboración de diez años entre el [Machine Learning Group - Universidad de Bruselas](https://mlg.ulb.ac.be/wordpress/) y la [empresa procesadora de pagos Worldline](https://uk.worldline.com/en/home.html).
