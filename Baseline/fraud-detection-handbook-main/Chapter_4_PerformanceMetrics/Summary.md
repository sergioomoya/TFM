# Resumen

Un sistema de detección de fraude tiene como objetivo optimizar dos objetivos contradictorios. Por un lado, el sistema optimizar la detección de transacciones fraudulentas, una métrica conocida como *recall* (recuperación). Por otro lado, también debe minimizar el número de falsos positivos, es decir, el número de transacciones genuinas que se predicen incorrectamente como fraudes. A menos que se pueda diseñar un clasificador perfecto, maximizar la recuperación tiene un costo en términos de falsos positivos. La estructura de costos de un sistema de detección de fraude es, sin embargo, compleja, si no imposible, de determinar. 

Un enfoque común para eludir este objetivo contradictorio es evaluar la recuperación y las tasas de falsos positivos para todos los umbrales de decisión posibles. La función resultante es la curva de Característica Operativa del Receptor (ROC). El área bajo la curva ROC (conocida como AUC ROC) es la métrica más utilizada para evaluar el rendimiento de los sistemas de detección de fraude. Este capítulo destacó que, sin embargo, el AUC ROC está mal adaptado a problemas de desequilibrio de clases, como la detección de fraude. Se motivaron otras dos métricas, la Precisión Promedio (AP) y la Card Precision diaria top-$k$ (CP@k), para caracterizar mejor el rendimiento de un CCFD. 

El AUC ROC, AP y CP@k son complementarios. El AUC ROC refleja la precisión del sistema de detección para todos los umbrales posibles. El AP también refleja la precisión del sistema de detección para todos los umbrales posibles, pero da más importancia a las regiones de los umbrales de decisión donde la precisión sigue siendo alta. Finalmente, el CP@k proporciona una métrica más concreta para CCFD, evaluando la precisión diaria promedio del sistema, asumiendo que los investigadores pueden verificar un máximo de $k$ tarjetas diariamente.  

Debido a su complementariedad, recomendamos evaluar el rendimiento de un sistema de fraude utilizando estas tres medidas de rendimiento. Todas las técnicas y métodos presentados en los próximos capítulos se evaluarán con estas medidas.
