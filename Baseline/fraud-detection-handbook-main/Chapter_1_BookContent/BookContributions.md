# Contribuciones del libro

Contribuciones originales del libro con respecto a la literatura actual:

* **Simulador de datos de transacciones sintéticas**: Este libro propone un simulador de datos de transacciones que permite la creación de conjuntos de datos de transacciones sintéticas de complejidad variable. En particular, el simulador permite variar el grado de desequilibrio de clases (baja proporción de transacciones fraudulentas), contiene variables tanto numéricas como categóricas (con características categóricas que tienen un número muy alto de valores posibles) y presenta escenarios de fraude dependientes del tiempo. El simulador se presenta en [Capítulo 3, Sección 2](Transaction_Data_Simulator).
* **Reproducibilidad**: Este libro es un [Jupyter Book](https://jupyterbook.org/intro.html), que permite ejecutar o modificar interactivamente las secciones de este libro que contienen código. Junto con el generador de datos sintéticos, todos los experimentos y resultados presentados en este libro son reproducibles. Se proporciona una descripción de cómo ejecutar este libro en la nube o en su computadora en [Capítulo 2, Sección 3](How_to_Use_This_Book).
* **Revisión del estado del arte**: El libro sintetiza las encuestas recientes sobre el tema del aprendizaje automático para la detección de fraude con tarjetas de crédito (ML para CCFD). Destaca los principios básicos presentados en estas encuestas y resume los principales desafíos de los sistemas de detección de fraude. La revisión se presenta en [Capítulo 3, sección 3](ML_For_CCFD).
* **Metodología de evaluación**: Una contribución importante de este libro es una presentación detallada y discusión de las métricas de rendimiento y metodologías de validación que se pueden utilizar para evaluar la eficiencia de los sistemas de detección de fraude. Las métricas de rendimiento se abordan en el [Capítulo 4](Performance_Metrics). Las metodologías de validación y las estrategias para la selección de modelos se abordan en el [Capítulo 5](Model_Validation_And_Selection).
* **Aprendizaje desequilibrado**: El libro proporciona una extensa evaluación experimental de enfoques de aprendizaje desequilibrado, que abarca técnicas sensibles al costo, remuestreo y conjunto (ensemble). Para cada enfoque, la evaluación experimental incluye un ejemplo de juguete, un conjunto de datos de transacciones simuladas y un conjunto de datos del mundo real. El tema se cubre en el [Capítulo 6](Imbalanced_Learning). La principal conclusión de la evaluación experimental propuesta es que los beneficios de las técnicas de aprendizaje desequilibrado son mitigados y dependen estrechamente de las métricas de rendimiento objetivo. 
* **Aprendizaje profundo**: El advenimiento reciente de técnicas de aprendizaje profundo ha llevado a la comunidad investigadora a interesarse cada vez más en sus aplicaciones para la detección de fraude. Este libro es el primero en entrar en los detalles del uso e implementación de este tipo de métodos para el problema de la detección de fraude con tarjetas de crédito. El [Capítulo 7](Deep_Learning) cubre la implementación y evaluación de técnicas como redes neuronales feed-forward totalmente conectadas, y técnicas más avanzadas como el aprendizaje de representación con autoencoders o modelos secuenciales como redes convolucionales o de memoria a largo plazo (LSTM).
